# Concepts

[Verona](https://verona.burnt.com/) is the intelligence layer for AI—the product narrative for the **Verona** network. It combines verified facts users control with protocol-level abstraction so apps and agents can act on trustworthy context.

Two foundational layers power the stack:

* **Truth Engine (Verification Infrastructure)** grounds intelligence in facts. Zero-knowledge proofs across web data (zkTLS), email (zkEmail/DKIM), and mobile apps (app attestations) let any internet-sourced claim be verified privately.
* **Generalized Abstraction** keeps the network usable. Protocol-level abstractions for accounts, signatures, gas, payments, devices, and interoperability remove barriers to mainstream adoption.
