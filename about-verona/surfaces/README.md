# Surfaces

Verona ships as a **network layer** plus **product surfaces**—interfaces where users and developers interact with verified intelligence. Month 1 documentation focuses on four surfaces that map to the intelligence-layer story.

Each surface builds on the same primitives documented under [Truth Engine](../concepts/verification-infrastructure/README.md) and [Build on Verona](../../developers/overview.md). Deep integration guides will expand as APIs and MCP servers stabilize.
